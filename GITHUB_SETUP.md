# GitHub-repositorion asetukset

## Projektin kuvaus GitHubissa

Aseta projektin kuvaus GitHub-repositorion "About" -osiossa:

1. Mene GitHub-repositorioon: https://github.com/5qtb5t9v5k-rgb/DocChamp
2. Klikkaa ⚙️ (asetukset-ikoni) "About" -osion vieressä
3. Täytä kentät:
   - **Description**: `DocChamp - Kuittien mestari`
   - **Website**: (jätä tyhjäksi tai lisää Streamlit Cloud -URL jos on)
   - **Topics**: `python`, `streamlit`, `ai`, `ocr`, `document-analysis`, `receipt-extraction`
4. Klikkaa "Save changes"

Tämän jälkeen projektin kuvaus näkyy repositorion etusivulla "About" -osiossa.
